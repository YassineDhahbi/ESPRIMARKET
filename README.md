# ESPRIMARKET
▪️ Développement d’une application multiplateforme pour la gestion des stocks, ventes et clients d’une épicerie. ▪️ Partie desktop réalisée avec Java, JavaFX et JDBC. ▪️ Partie web développée avec Symfony 5.4, offrant une gestion en ligne . ▪️ Partie mobile créée avec FlutterFlow + Firebase ▪️ Utilisation de MySQL pour centraliser données
